# **Note-Taking-App**

### A note taking app. <br/> <br/>

## **Functions**

### Notes can be added, updated and deleted. All notes are showed order by time in a recyclerview. <br/> <br/>

## **Used**

### MVVM, SQLite-Room, Live Data, Data Binding, Coroutines <br/> <br/>
